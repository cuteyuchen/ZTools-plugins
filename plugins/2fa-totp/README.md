# 2FA TOTP 插件

## 简介
这是一个为 ZTools 平台开发的 2FA 动态验证码管理插件。它提供安全、极速、美观的多协议双重认证管理功能。

## 特性
- **全协议支持**：完美支持 TOTP 基于时间、HOTP 基于计数器以及 Steam 令牌自定义 Base26 算法。
- **高级配置**：
    - 支持多种哈希算法：SHA1、SHA256、SHA512。
    - 支持自定义验证码位数：5 位 Steam、6 位、7 位、8 位。
    - 支持自定义时间周期：15 秒、30 秒、60 秒或手动计数器。
- **极致交互**：
    - **下一个验证码预览**：支持在主界面实时显示下一个周期的验证码，可配置开关。
    - **智能联动**：在选择 Steam 协议时，自动锁定并填充协议规范为 SHA1 算法、5 位数字、30 秒周期。
    - **自动清洗**：密钥输入时自动转换为大写并去除空格，提升录入体验。
    - **密钥合法性验证**：添加账号时自动进行 Base32 解码校验，确保密钥格式正确。
    - **同步验证反馈**：账号名称和密钥同时验证，错误提示对称显示在输入框两侧。
    - **HOTP 节流刷新**：专为计数型协议设计的带 5 秒冷却保护的刷新按钮。
    - **视觉反馈**：全域点击快捷复制，复制成功后 2 秒勾号提示。
- **现代 UI**：网格化配置界面，所有垂直菜单均采用上拉模式适配小窗；支持指示器模式切换为环形或条形。
