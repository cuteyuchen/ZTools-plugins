# 2FA TOTP 插件 (2fa-totp)

## 简介
这是一个为 ZTools 平台开发的 2FA TOTP/HOTP 动态验证码管理插件。它提供安全、极速、美观的双重认证验证码管理功能。

## 特性
- **全协议支持**：完美支持 **TOTP**（时间型）与 **HOTP**（计数型）协议。
- **高级配置**：
    - 支持多种哈希算法：SHA1, SHA256, SHA512。
    - 支持自定义验证码位数：6, 7, 8 位。
    - 支持自定义时间周期（15/30/60s）或手动计数器。
- **极致交互**：
    - **HOTP 5s 节流刷新**：专为计数型协议设计的带冷却保护的刷新按钮。
    - **全域点击复制**：支持点击卡片自动复制及自动输入功能。
    - **视觉反馈**：复制成功后 2s 勾号提示；低时段精准变色预警。
- **现代 UI**：网格化 2 列配置界面，支持 Drop-up 上拉菜单，适配小窗显示。
- **智能校验**：Base32 编码合法性实时校验，秒级瞬时报错动画。

## 安装与开发
- 依赖项：仅需加载 `vue.global.js`。
- 入口文件：`index.html`。
- 逻辑层：`app.js`。
- 预加载：`preload.js`。
